# Materiais de apoio

Esta pasta guarda arquivos editaveis e originais que ajudam na manutencao do projeto, mas nao sao carregados pelo site em producao.

## Pastas

- `google-business-cover/`: arquivos editaveis da capa/arte para Google Business.
- `originals/`: fotos originais ou alternativas que nao estao sendo usadas diretamente no HTML.

Os arquivos essenciais do site ficam na raiz e em `assets/`.
